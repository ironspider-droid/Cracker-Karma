# **App Name**: Cracker Karma

## Core Features:

- Pollution Tracking: Track the user's daily activities and carbon footprint through sensor data from gadgets and fitness bands.
- Cracker Budget Calculation: Calculate a personalized cracker budget for the next year based on the user's pollution data.
- Reward System: Implement a reward system where users earn credits for eco-friendly activities like planting trees, which can be redeemed to increase their cracker budget.
- Cracker Analysis Tool: Allow users to upload photos of crackers, and use an AI tool to estimate the amount of pollution each cracker will cause and update the budget accordingly.
- Budget Warning System: Implement a warning system that alerts users when their cracker budget is exceeded and provides suggestions for increasing the budget through eco-friendly activities.
- Cracker Category Guide: Display information on how many crackers of each category the user is allowed to burst based on their budget.

## Style Guidelines:

- Primary color: Soft green (#A7D1AB) to evoke a sense of nature and environmental consciousness.
- Background color: Very light green (#F0F4F1), close to white, with just a hint of the primary color's hue.
- Accent color: Warm yellow (#E5B83B) to highlight rewards and budget increases.
- Body and headline font: 'PT Sans', a modern-looking humanist sans-serif, for readability.
- Use cute, simple icons related to environmental activities and cracker categories.
- Design a user-friendly interface with clear, intuitive navigation.
- Incorporate subtle animations for rewards and budget updates to enhance user engagement.
